# CEUPR website

## Sobre

Site feito como sugestão de website para a Casa do Estudante do Paraná.

## Uso

Use o gerenciador de pacote [npm](https://docs.npmjs.com/) para instalar os pacotes usados no projeto.

```bash
npm install
```

Execute para rodar o projeto

```bash
npm start
```
